EJERCICIO MULTIPROCESSING CON POOL

Este programa calcula la suma de los números del 1 a N usando un Pool de procesos.

Archivos:
- suma.py: contiene la función que ejecutan los procesos del Pool.
- main.py: crea el Pool con distinto número de procesos y compara los tiempos.
- README.txt: información del ejercicio.

Funcionamiento:
El programa lanza la misma tarea con diferente número de procesos (1, 2, 4 y 8)
y mide cuánto tiempo tarda en cada caso para comparar el rendimiento.

Ejecución:
python main.py
